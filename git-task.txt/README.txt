# Git Task

## Description
This project contains practical exercises for learning Git and basic file/folder management using scripts.  
It demonstrates:

- Initializing a Git repository
- Tracking and staging files
- Committing changes
- Viewing Git logs
- Basic Bash/PowerShell scripting for creating files and folders
- Using conditional statements in scripts (if, if-else, nested if)

## Files Included
- `hello_world.py` or `hello_world.js` – Example program demonstrating file creation and modification.
- `file_cd.ps1` – PowerShell script creating folders, navigating directories, and removing folders.
- `ifExample.ps1` – PowerShell script demonstrating conditional folder creation.
- `example-repo.txt` – Contains the URL of the public GitHub repository.
- `README.md` – Project description.
